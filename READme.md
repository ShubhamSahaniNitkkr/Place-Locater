![Screenshot](./screenshot.png)

## Purpose

Find nearby stores on an interactive map using geolocation and MongoDB.

## Use Cases

- Map-based store finder
- Geospatial queries
- Location API integration

## Tech Stack

- Node.js
- Express
- MongoDB
- Mapbox
- Geocoder

## How to Run Locally

```bash
npm install && npm run demo
```
